# FitTrack Pro - Product Landing Page

This project is a responsive product landing page for **FitTrack Pro**, a smart fitness tracker designed to help users monitor their health and improve their fitness journey.

Features
- Responsive and modern design
- Interactive navigation bar
- Smooth scrolling between sections
- Countdown timer for limited offers
- Modal popup for purchase confirmation
- Animated feature cards and pricing section

Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

Project Structure
product-landing-page/
│── index.html
│── styles.css
│── script.js
│── README.md

How to Run
1. Download or clone the repository  
2. Open `index.html` in your browser  

Purpose
This project was created as part of a web development activity to demonstrate:
- Frontend design skills
- User interaction using JavaScript
- Clean and structured code organization
